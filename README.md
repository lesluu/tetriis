Классическая игра Тетрис, реализованная на C# с использованием Windows Forms. Проект включает основные функции оригинальной игры.

Особенности
Классический игровой процесс Тетрис
Hold фигура (удержание фигуры)
Сохранение рекорда между сессиями
Два типа падения: мягкое и жесткое

Статистика игры
Анимированные бонусные сообщения
Адаптивная скорость игры

 Управление
Клавиша	Действие
← →	Движение фигуры влево/вправо
↑	Поворот фигуры
↓	Мягкое падение (+1 очко за клетку)
Пробел	Жесткое падение (+2 очка за клетку)
C	Hold фигура (удержать текущую фигуру)
R	Рестарт игры

Система подсчета очков
Обычные линии: 100 очков за линию
Тетрис (4 линии): 800 очков (бонус!)
Мягкое падение: +1 очко за каждую клетку
Жесткое падение: +2 очка за каждую клетку

Рекорд: Сохраняется в файл highscore.txt

Архитектура проекта
Основные классы:
Form1 - Главная форма игры
Обработка ввода
Отрисовка интерфейса
Управление таймерами

GameBoard - Игровая логика
Управление игровым полем
Обработка столкновений
Подсчет очков
Управление фигурами
Сохранение рекорда

Tetromino - Фигура Тетриса
7 классических форм
Система вращения
Цветовое кодирование

Program - Точка входа приложения

Особенности реализации
Hold система:
Можно удерживать только одну фигуру
Можно использовать Hold один раз за фигуру
Визуальное отображение удерживаемой фигуры
Возможность обмена текущей и удерживаемой фигур

Система сохранения:
Рекорд сохраняется в текстовом файл
Автоматическая загрузка при старте
Автоматическое сохранение при обновлении

Визуальные эффекты:
Плавное исчезновение бонусных сообщений
Разные цвета для каждой фигуры
Сетка игрового поля
Предпросмотр следующей фигуры

Технические детали
Игровое поле: 10×20 клеток
Размер клетки: 30×30 пикселей
Таймер игры: 500 мс
Поддерживаемые фигуры: I, J, L, O, S, T, Z
Цветовая схема: Классические цвета Тетриса
